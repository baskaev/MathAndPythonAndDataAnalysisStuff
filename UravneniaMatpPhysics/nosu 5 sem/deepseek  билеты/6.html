Билет № 6: Уравнения математической физики

1. Общая краевая задача для уравнений колебаний

Что такое краевая задача?

Краевая задача — это задача, в которой ищется решение уравнения с частными производными, удовлетворяющее определенным условиям на границах области.

Уравнение колебаний:

\[ \frac{\partial^2 u}{\partial t^2} = c^2 \left( \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} + \frac{\partial^2 u}{\partial z^2} \right), \]

где:

  • \( u(x, y, z, t) \) — отклонение от положения равновесия,
  • \( c \) — скорость распространения волн.

Граничные условия:

  1. Закрепленные границы:

    \[ u(0, t) = 0, \quad u(L, t) = 0. \]

  2. Свободные границы:

    \[ \frac{\partial u}{\partial x}(0, t) = 0, \quad \frac{\partial u}{\partial x}(L, t) = 0. \]

Решение методом разделения переменных:

  1. Предположим, что решение имеет вид:

    \[ u(x, t) = X(x) T(t). \]

  2. Подставим в уравнение:

    \[ X(x) T''(t) = c^2 X''(x) T(t). \]

  3. Разделим на \( X(x) T(t) \):

    \[ \frac{T''(t)}{c^2 T(t)} = \frac{X''(x)}{X(x)} = -\lambda. \]

  4. Получаем два обыкновенных дифференциальных уравнения:
    • Для \( X(x) \):

      \[ X''(x) + \lambda X(x) = 0. \]

    • Для \( T(t) \):

      \[ T''(t) + \lambda c^2 T(t) = 0. \]

  5. Решим уравнение для \( X(x) \): Общее решение:

    \[ X(x) = A \sin(\sqrt{\lambda} x) + B \cos(\sqrt{\lambda} x). \]

    Из граничных условий \( X(0) = 0 \) и \( X(L) = 0 \) получаем:

    \[ \sqrt{\lambda} = \frac{n \pi}{L}, \quad n = 1, 2, 3, \dots \]

    Таким образом:

    \[ X_n(x) = \sin\left(\frac{n \pi x}{L}\right). \]

  6. Решим уравнение для \( T(t) \): Общее решение:

    \[ T_n(t) = C_n \cos\left(\frac{n \pi c t}{L}\right) + D_n \sin\left(\frac{n \pi c t}{L}\right). \]

  7. Общее решение уравнения колебаний:

    \[ u(x, t) = \sum_{n=1}^{\infty} \left[ C_n \cos\left(\frac{n \pi c t}{L}\right) + D_n \sin\left(\frac{n \pi c t}{L}\right) \right] \sin\left(\frac{n \pi x}{L}\right). \]

2. Уравнение Лапласа в полярной системе координат

Что такое уравнение Лапласа?

Уравнение Лапласа описывает стационарные процессы, например, распределение температуры в равновесии или потенциал электростатического поля. В двумерном случае оно имеет вид:

\[ \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0. \]

Переход к полярным координатам:

В полярных координатах \( (r, \theta) \) уравнение Лапласа принимает вид:

\[ \frac{\partial^2 u}{\partial r^2} + \frac{1}{r} \frac{\partial u}{\partial r} + \frac{1}{r^2} \frac{\partial^2 u}{\partial \theta^2} = 0. \]

Решение методом разделения переменных:

  1. Предположим, что решение имеет вид:

    \[ u(r, \theta) = R(r) \Theta(\theta). \]

  2. Подставим в уравнение:

    \[ R''(r) \Theta(\theta) + \frac{1}{r} R'(r) \Theta(\theta) + \frac{1}{r^2} R(r) \Theta''(\theta) = 0. \]

  3. Разделим на \( R(r) \Theta(\theta) \):

    \[ \frac{R''(r)}{R(r)} + \frac{1}{r} \frac{R'(r)}{R(r)} + \frac{1}{r^2} \frac{\Theta''(\theta)}{\Theta(\theta)} = 0. \]

  4. Получаем два обыкновенных дифференциальных уравнения:
    • Для \( \Theta(\theta) \):

      \[ \Theta''(\theta) + \lambda \Theta(\theta) = 0. \]

    • Для \( R(r) \):

      \[ r^2 R''(r) + r R'(r) - \lambda R(r) = 0. \]

  5. Решим уравнение для \( \Theta(\theta) \): Общее решение:

    \[ \Theta(\theta) = A \cos(\sqrt{\lambda} \theta) + B \sin(\sqrt{\lambda} \theta). \]

    Из условия периодичности \( \Theta(\theta + 2\pi) = \Theta(\theta) \) получаем:

    \[ \sqrt{\lambda} = n, \quad n = 0, 1, 2, \dots \]

    Таким образом:

    \[ \Theta_n(\theta) = A_n \cos(n \theta) + B_n \sin(n \theta). \]

  6. Решим уравнение для \( R(r) \): Общее решение:

    \[ R_n(r) = C_n r^n + D_n r^{-n}. \]

  7. Общее решение уравнения Лапласа в полярных координатах:

    \[ u(r, \theta) = \sum_{n=0}^{\infty} \left( C_n r^n + D_n r^{-n} \right) \left( A_n \cos(n \theta) + B_n \sin(n \theta) \right). \]
