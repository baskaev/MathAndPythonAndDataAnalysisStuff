1. Классификация уравнений: Гиперболические (волны), параболические (тепло), эллиптические (Лаплас).
2. Канонические формы: Гиперболическое: uxx - utt = 0, параболическое: ut - uxx = 0, эллиптическое: uxx + uyy = 0.
3. Уравнение струны: utt = a2 uxx, решение: u(x,t) = X(x)T(t).
4. Уравнение теплопроводности: ut = a2 uxx, решение: u(x,t) = X(x)T(t).
5. Уравнение Лапласа: Δu = 0, в полярных координатах: u(r,θ) = Σ (An rn + Bn r-n)(Cn cos(nθ) + Dn sin(nθ)).
6. Задача Коши: Для струны: u(x,0) = φ(x), ut(x,0) = ψ(x). Для тепла: u(x,0) = φ(x).
7. Краевые задачи: Дирихле (значения на границе), Неймана (производные на границе), смешанная.
8. Метод разделения переменных: u(x,t) = X(x)T(t).
9. Фундаментальные решения: Лаплас: Φ(x) = 1/|x|n-2, тепло: Φ(x,t) = 1/(4πt)n/2 e-|x|2/4t.
10. Принцип максимума: Параболические: максимум на границе/начале. Эллиптические: максимум/минимум на границе.
11. Задача Дирихле в круге: Решение: u(r,θ) = Σ (An rn cos(nθ) + Bn rn sin(nθ)).
12. Формула Пуассона: u(r,θ) = (1/2π) ∫02π (R2 - r2) / (R2 - 2Rr cos(θ - φ) + r2) f(φ) dφ.
13. Теорема Гаусса-Остроградского: ∫∂V F · dS = ∫V (∇ · F) dV.
14. Метод спуска: Сведение многомерных задач к одномерным.
15. Устойчивость: Малые изменения начальных данных → малые изменения решения.
16. Сферические и цилиндрические координаты: Используются для задач с симметрией.
17. Теорема о среднем: Значение гармонической функции в центре сферы = среднее на поверхности.
18. Задача Гурса: Данные на характеристиках для гиперболических уравнений.
19. Методы ТФКП: Решение краевых задач с конформными отображениями.
20. Неоднородные уравнения: Решение через вариацию постоянных или интеграл Дюамеля.
