Билет № 7: Уравнения математической физики

1. Задача Гурса с данными на характеристиках

Что такое задача Гурса?

Задача Гурса — это задача, в которой ищется решение уравнения с частными производными, удовлетворяющее заданным условиям на характеристиках.

Что такое характеристика?

Характеристика — это кривая, вдоль которой решение уравнения с частными производными может быть выражено через начальные данные. Для гиперболических уравнений характеристики играют важную роль, так как вдоль них информация распространяется.

Уравнение с характеристиками:

Для гиперболического уравнения второго порядка:

\[ A(x, y) \frac{\partial^2 u}{\partial x^2} + 2B(x, y) \frac{\partial^2 u}{\partial x \partial y} + C(x, y) \frac{\partial^2 u}{\partial y^2} = 0, \]

характеристики определяются уравнением:

\[ A(dy)^2 - 2B dx dy + C(dx)^2 = 0. \]

Пример: Волновое уравнение:

Рассмотрим волновое уравнение:

\[ \frac{\partial^2 u}{\partial t^2} = c^2 \frac{\partial^2 u}{\partial x^2}. \]

Характеристики этого уравнения:

\[ x - ct = \text{const}, \quad x + ct = \text{const}. \]

Задача Гурса:

Задача Гурса заключается в нахождении решения уравнения, удовлетворяющего заданным условиям на характеристиках. Например, для волнового уравнения:

\[ u(x, 0) = f(x), \quad \frac{\partial u}{\partial t}(x, 0) = g(x). \]

Решение задачи Гурса:

Решение задачи Гурса для волнового уравнения можно найти с помощью формулы Даламбера:

\[ u(x, t) = \frac{1}{2} \left[ f(x - ct) + f(x + ct) \right] + \frac{1}{2c} \int_{x - ct}^{x + ct} g(s) ds. \]

Пример:

Пусть начальное отклонение струны задано функцией:

\[ f(x) = \sin\left(\frac{\pi x}{L}\right), \]

а начальная скорость равна нулю:

\[ g(x) = 0. \]

Тогда решение задачи Коши:

\[ u(x, t) = \frac{1}{2} \left[ \sin\left(\frac{\pi (x - ct)}{L}\right) + \sin\left(\frac{\pi (x + ct)}{L}\right) \right]. \]

2. Фундаментальное решение для уравнения Лапласа

Что такое фундаментальное решение?

Фундаментальное решение — это решение уравнения с частными производными, которое позволяет находить решения для произвольных начальных условий.

Уравнение Лапласа:

Уравнение Лапласа в двумерном случае:

\[ \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0. \]

Фундаментальное решение:

Фундаментальное решение уравнения Лапласа в двумерном случае:

\[ G(x, y) = \frac{1}{2\pi} \ln \sqrt{x^2 + y^2}. \]

Пример использования:

Фундаментальное решение используется для нахождения решения уравнения Лапласа с заданными граничными условиями. Например, для задачи Дирихле:

\[ u(x, y) = \int_{\partial \Omega} G(x - \xi, y - \eta) f(\xi, \eta) dS, \]

где \( f(\xi, \eta) \) — граничные условия.
