Билет № 3: Уравнения математической физики

1. Задачи колебаний стержня

Что такое стержень?

Стержень — это длинный, тонкий, упругий объект, который может колебаться. Колебания стержня описываются волновым уравнением, аналогично колебаниям струны, но с учетом поперечных или продольных деформаций.

Физические предположения:

  1. Стержень однородный (плотность \( \rho \) постоянна).
  2. Модуль упругости \( E \) постоянен.
  3. Отклонения стержня малы, поэтому углы наклона малы.

Уравнение колебаний стержня:

\[ \frac{\partial^2 u}{\partial t^2} = c^2 \frac{\partial^4 u}{\partial x^4}, \]

где:

  • \( u(x, t) \) — отклонение стержня от положения равновесия,
  • \( c = \sqrt{\frac{E}{\rho}} \) — скорость распространения волн в стержне.

Граничные условия:

  • Закрепленный конец:

    \[ u(0, t) = 0, \quad \frac{\partial u}{\partial x}(0, t) = 0. \]

  • Свободный конец:

    \[ \frac{\partial^2 u}{\partial x^2}(L, t) = 0, \quad \frac{\partial^3 u}{\partial x^3}(L, t) = 0. \]

Решение методом разделения переменных:

Предположим, что решение имеет вид:

\[ u(x, t) = X(x) T(t). \]

Подставим в уравнение:

\[ X(x) T''(t) = c^2 X''''(x) T(t). \]

Разделим на \( X(x) T(t) \):

\[ \frac{T''(t)}{c^2 T(t)} = \frac{X''''(x)}{X(x)} = -\lambda. \]

Получаем два обыкновенных дифференциальных уравнения:

  1. Для \( X(x) \):

    \[ X''''(x) + \lambda X(x) = 0. \]

  2. Для \( T(t) \):

    \[ T''(t) + \lambda c^2 T(t) = 0. \]

Решив эти уравнения, можно найти общее решение.

2. Краевые задачи без начальных условий

Что такое краевая задача?

Краевая задача — это задача, в которой ищется решение уравнения с частными производными, удовлетворяющее определенным условиям на границах области. В случае без начальных условий, задача решается только на основе граничных условий.

Пример: Уравнение Лапласа:

\[ \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0. \]

Граничные условия:

  1. На левой границе (\( x = 0 \)):

    \[ u(0, y) = f(y). \]

  2. На правой границе (\( x = L \)):

    \[ u(L, y) = g(y). \]

  3. На нижней границе (\( y = 0 \)):

    \[ u(x, 0) = h(x). \]

  4. На верхней границе (\( y = H \)):

    \[ u(x, H) = k(x). \]

Решение методом разделения переменных:

Предположим, что решение имеет вид:

\[ u(x, y) = X(x) Y(y). \]

Подставим в уравнение Лапласа:

\[ X''(x) Y(y) + X(x) Y''(y) = 0. \]

Разделим на \( X(x) Y(y) \):

\[ \frac{X''(x)}{X(x)} = -\frac{Y''(y)}{Y(y)} = -\lambda. \]

Получаем два обыкновенных дифференциальных уравнения:

  1. Для \( X(x) \):

    \[ X''(x) + \lambda X(x) = 0. \]

  2. Для \( Y(y) \):

    \[ Y''(y) - \lambda Y(y) = 0. \]

Решив эти уравнения, можно найти общее решение.

Итог

1. Задачи колебаний стержня:

  • Уравнение колебаний стержня: \( \frac{\partial^2 u}{\partial t^2} = c^2 \frac{\partial^4 u}{\partial x^4} \).
  • Граничные условия.
  • Решение методом разделения переменных.

2. Краевые задачи без начальных условий:

  • Уравнение Лапласа: \( \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0 \).
  • Граничные условия.
  • Решение методом разделения переменных.
