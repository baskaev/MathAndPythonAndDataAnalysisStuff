Билет № 5: Уравнения математической физики

1. Уравнение распространения тепла в изотропном твердом теле

Что такое изотропное твердое тело?

Изотропное твердое тело — это тело, свойства которого (например, теплопроводность) одинаковы во всех направлениях.

Уравнение теплопроводности:

\[ \frac{\partial u}{\partial t} = \alpha \left( \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} + \frac{\partial^2 u}{\partial z^2} \right), \]

где:

  • \( u(x, y, z, t) \) — температура в точке \( (x, y, z) \) в момент времени \( t \),
  • \( \alpha \) — коэффициент теплопроводности.

Физический смысл уравнения:

  1. Левый член \( \frac{\partial u}{\partial t} \): Скорость изменения температуры в точке.
  2. Правый член \( \alpha \left( \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} + \frac{\partial^2 u}{\partial z^2} \right) \): Описывает диффузию тепла в теле. Чем больше вторая производная температуры по координатам, тем быстрее тепло распространяется.

Пример: Одномерное уравнение теплопроводности:

Рассмотрим одномерное уравнение теплопроводности:

\[ \frac{\partial u}{\partial t} = \alpha \frac{\partial^2 u}{\partial x^2}. \]

Граничные условия:

  1. На левой границе (\( x = 0 \)):

    \[ u(0, t) = T_0. \]

  2. На правой границе (\( x = L \)):

    \[ u(L, t) = T_1. \]

Начальное условие:

\[ u(x, 0) = f(x). \]

Решение методом разделения переменных:

  1. Предположим, что решение имеет вид:

    \[ u(x, t) = X(x) T(t). \]

  2. Подставим в уравнение:

    \[ X(x) T'(t) = \alpha X''(x) T(t). \]

  3. Разделим на \( X(x) T(t) \):

    \[ \frac{T'(t)}{\alpha T(t)} = \frac{X''(x)}{X(x)} = -\lambda. \]

  4. Получаем два обыкновенных дифференциальных уравнения:
    • Для \( X(x) \):

      \[ X''(x) + \lambda X(x) = 0. \]

    • Для \( T(t) \):

      \[ T'(t) + \alpha \lambda T(t) = 0. \]

  5. Решим уравнение для \( X(x) \): Общее решение:

    \[ X(x) = A \sin(\sqrt{\lambda} x) + B \cos(\sqrt{\lambda} x). \]

    Из граничных условий \( X(0) = 0 \) и \( X(L) = 0 \) получаем:

    \[ \sqrt{\lambda} = \frac{n \pi}{L}, \quad n = 1, 2, 3, \dots \]

    Таким образом:

    \[ X_n(x) = \sin\left(\frac{n \pi x}{L}\right). \]

  6. Решим уравнение для \( T(t) \): Общее решение:

    \[ T_n(t) = C_n e^{-\alpha \left(\frac{n \pi}{L}\right)^2 t}. \]

  7. Общее решение уравнения теплопроводности:

    \[ u(x, t) = \sum_{n=1}^{\infty} C_n \sin\left(\frac{n \pi x}{L}\right) e^{-\alpha \left(\frac{n \pi}{L}\right)^2 t}. \]

  8. Определим коэффициенты \( C_n \): Используем начальное условие \( u(x, 0) = f(x) \):

    \[ f(x) = \sum_{n=1}^{\infty} C_n \sin\left(\frac{n \pi x}{L}\right). \]

    Коэффициенты \( C_n \) находятся по формуле:

    \[ C_n = \frac{2}{L} \int_0^L f(x) \sin\left(\frac{n \pi x}{L}\right) dx. \]

2. Задача Коши для уравнения колебаний струны

Что такое задача Коши?

Задача Коши — это задача, в которой ищется решение уравнения с частными производными, удовлетворяющее начальным условиям.

Уравнение колебаний струны:

\[ \frac{\partial^2 u}{\partial t^2} = c^2 \frac{\partial^2 u}{\partial x^2}, \]

где:

  • \( u(x, t) \) — отклонение струны от положения равновесия,
  • \( c \) — скорость распространения волн в струне.

Начальные условия:

  1. Начальное отклонение струны:

    \[ u(x, 0) = f(x). \]

  2. Начальная скорость струны:

    \[ \frac{\partial u}{\partial t}(x, 0) = g(x). \]

Решение методом Даламбера:

Решение задачи Коши для волнового уравнения можно найти с помощью формулы Даламбера:

\[ u(x, t) = \frac{1}{2} \left[ f(x - ct) + f(x + ct) \right] + \frac{1}{2c} \int_{x - ct}^{x + ct} g(s) ds. \]

Пример:

Пусть начальное отклонение струны задано функцией:

\[ f(x) = \sin\left(\frac{\pi x}{L}\right), \]

а начальная скорость равна нулю:

\[ g(x) = 0. \]

Тогда решение задачи Коши:

\[ u(x, t) = \frac{1}{2} \left[ \sin\left(\frac{\pi (x - ct)}{L}\right) + \sin\left(\frac{\pi (x + ct)}{L}\right) \right]. \]

Итог

1. Уравнение распространения тепла в изотропном твердом теле:

  • Уравнение: \( \frac{\partial u}{\partial t} = \alpha \left( \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} + \frac{\partial^2 u}{\partial z^2} \right) \).
  • Решение методом разделения переменных.

2. Задача Коши для уравнения колебаний струны:

  • Уравнение: \( \frac{\partial^2 u}{\partial t^2} = c^2 \frac{\partial^2 u}{\partial x^2} \).
  • Решение с помощью формулы Даламбера.
