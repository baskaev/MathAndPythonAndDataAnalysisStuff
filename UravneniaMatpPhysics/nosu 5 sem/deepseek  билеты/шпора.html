Шпаргалка по дифференциальным уравнениям

1. Основные типы дифференциальных уравнений

  • Обыкновенные дифференциальные уравнения (ОДУ): Уравнения, содержащие одну независимую переменную и производные искомой функции по этой переменной.
  • Дифференциальные уравнения в частных производных (УРЧП): Уравнения, содержащие несколько независимых переменных и частные производные искомой функции.
  • Линейные дифференциальные уравнения: Уравнения, в которых искомая функция и её производные входят линейно.
  • Неоднородные дифференциальные уравнения: Уравнения, содержащие ненулевую функцию в правой части.

2. Методы решения ОДУ

  • Метод разделения переменных: Применяется для уравнений вида dy/dx = f(x)g(y).
  • Метод интегрирующего множителя: Используется для линейных уравнений первого порядка.
  • Метод вариации постоянных: Применяется для решения неоднородных линейных уравнений.
  • Метод характеристик: Используется для решения линейных уравнений в частных производных.

3. Пример решения дифференциального уравнения

Задача: Решить дифференциальное уравнение dy/dx + y = e^x.

Решение:

  1. Запишем уравнение в стандартной форме: dy/dx + y = e^x.
  2. Найдем интегрирующий множитель: μ(x) = e^(∫1 dx) = e^x.
  3. Умножим обе части уравнения на интегрирующий множитель: e^x dy/dx + e^x y = e^(2x).
  4. Левая часть уравнения является производной произведения: d/dx (e^x y) = e^(2x).
  5. Проинтегрируем обе части: e^x y = ∫e^(2x) dx = (1/2)e^(2x) + C.
  6. Найдем общее решение: y = (1/2)e^x + C e^(-x).

Ответ: y = (1/2)e^x + C e^(-x), где C — произвольная постоянная.

4. Полезные формулы

  • Линейное ОДУ первого порядка: dy/dx + P(x)y = Q(x).
  • Интегрирующий множитель: μ(x) = e^(∫P(x) dx).
  • Общее решение линейного ОДУ: y = (1/μ(x)) ∫μ(x)Q(x) dx + C.
