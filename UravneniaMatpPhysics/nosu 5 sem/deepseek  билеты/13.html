Билет № 13: Уравнения математической физики

1. Начально-краевая задача для полуограниченного стержня

Определение:

Полуограниченный стержень — это стержень, который простирается от \( x = 0 \) до \( x = \infty \). Начально-краевая задача для такого стержня включает начальное распределение температуры и граничные условия на конце \( x = 0 \).

Уравнение теплопроводности:

Уравнение, описывающее распределение температуры \( u(x, t) \) в стержне, имеет вид: \[ u_t = a^2 u_{xx}, \] где:

  • \( u(x, t) \) — температура в точке \( x \) в момент времени \( t \),
  • \( a \) — коэффициент теплопроводности.

Начальное условие:

Задано начальное распределение температуры: \[ u(x, 0) = \phi(x). \]

Граничное условие:

На конце стержня \( x = 0 \) задано условие, например: \[ u(0, t) = \mu(t). \]

Решение:

1. Метод функции Грина: Для полуограниченного стержня решение можно найти с помощью функции Грина \( G(x, t; \xi, \tau) \), которая описывает влияние точечного источника тепла в точке \( \xi \) в момент времени \( \tau \) на температуру в точке \( x \) в момент времени \( t \). 2. Функция Грина для полуограниченного стержня с граничным условием \( u(0, t) = 0 \) имеет вид: \[ G(x, t; \xi, \tau) = \frac{1}{2a \sqrt{\pi (t - \tau)}} \left( e^{-\frac{(x - \xi)^2}{4a^2 (t - \tau)}} - e^{-\frac{(x + \xi)^2}{4a^2 (t - \tau)}} \right). \] 3. Решение задачи Коши с нулевым граничным условием: \[ u(x, t) = \int_0^{\infty} \phi(\xi) G(x, t; \xi, 0) \, d\xi. \] 4. Если граничное условие \( u(0, t) = \mu(t) \), то решение можно найти с помощью метода отражений или интеграла Дюамеля.

Пример задачи:

Решить начально-краевую задачу для полуограниченного стержня с начальным условием \( \phi(x) = 0 \) и граничным условием \( u(0, t) = T_0 \) (постоянная температура на конце стержня).

Решение: 1. Используем метод отражений. Решение имеет вид: \[ u(x, t) = T_0 \left( 1 - \text{erf}\left( \frac{x}{2a \sqrt{t}} \right) \right), \] где \( \text{erf}(z) \) — функция ошибок, определяемая как: \[ \text{erf}(z) = \frac{2}{\sqrt{\pi}} \int_0^z e^{-t^2} dt. \] 2. Это решение описывает, как температура распространяется по стержню со временем, начиная с постоянной температуры \( T_0 \) на конце \( x = 0 \).

2. Вывод формулы Пуассона для круга

Определение:

Формула Пуассона позволяет найти решение задачи Дирихле для уравнения Лапласа в круге. Она выражает значение функции \( u(r, \theta) \) внутри круга через её значения на границе.

Уравнение Лапласа в полярных координатах:

\[ \Delta u = u_{rr} + \frac{1}{r} u_r + \frac{1}{r^2} u_{\theta \theta} = 0. \]

Граничное условие:

\[ u(R, \theta) = f(\theta), \] где \( f(\theta) \) — заданная функция на границе круга радиуса \( R \).

Формула Пуассона:

\[ u(r, \theta) = \frac{1}{2\pi} \int_0^{2\pi} \frac{R^2 - r^2}{R^2 - 2R r \cos(\theta - \phi) + r^2} f(\phi) \, d\phi. \]

Вывод формулы Пуассона:

1. Решение уравнения Лапласа в круге можно представить в виде ряда Фурье: \[ u(r, \theta) = \sum_{n=0}^{\infty} r^n \left( A_n \cos(n \theta) + B_n \sin(n \theta) \right). \] 2. Коэффициенты \( A_n \) и \( B_n \) находятся из граничного условия: \[ f(\theta) = \sum_{n=0}^{\infty} R^n \left( A_n \cos(n \theta) + B_n \sin(n \theta) \right). \] 3. Используя свойства рядов Фурье и интегральные представления, можно показать, что решение можно записать в виде интеграла Пуассона: \[ u(r, \theta) = \frac{1}{2\pi} \int_0^{2\pi} \frac{R^2 - r^2}{R^2 - 2R r \cos(\theta - \phi) + r^2} f(\phi) \, d\phi. \]

Пример задачи:

Найти решение задачи Дирихле в круге радиуса \( R = 1 \) с граничным условием \( f(\theta) = \cos(\theta) \).

Решение: 1. Подставляем \( f(\theta) = \cos(\theta) \) в формулу Пуассона: \[ u(r, \theta) = \frac{1}{2\pi} \int_0^{2\pi} \frac{1 - r^2}{1 - 2r \cos(\theta - \phi) + r^2} \cos(\phi) \, d\phi. \] 2. Используя свойства интегралов и тригонометрические тождества, можно показать, что: \[ u(r, \theta) = r \cos(\theta). \]

Вопросы, которые могут возникнуть у преподавателя, и ответы на них:

Вопрос 1: Что такое начально-краевая задача для полуограниченного стержня?

Ответ:

Это задача нахождения распределения температуры в полуограниченном стержне, которая включает начальное распределение температуры и граничные условия на конце стержня.

Вопрос 2: Как решается задача для полуограниченного стержня с постоянной температурой на конце?

Ответ:

Решение можно найти с помощью метода отражений или функции ошибок. Решение имеет вид: \[ u(x, t) = T_0 \left( 1 - \text{erf}\left( \frac{x}{2a \sqrt{t}} \right) \right). \]

Вопрос 3: Что такое формула Пуассона для круга?

Ответ:

Формула Пуассона позволяет найти решение задачи Дирихле для уравнения Лапласа в круге. Она выражает значение функции внутри круга через её значения на границе: \[ u(r, \theta) = \frac{1}{2\pi} \int_0^{2\pi} \frac{R^2 - r^2}{R^2 - 2R r \cos(\theta - \phi) + r^2} f(\phi) \, d\phi. \]

Вопрос 4: Как выводится формула Пуассона?

Ответ:

Формула Пуассона выводится с помощью представления решения уравнения Лапласа в виде ряда Фурье и использования интегральных представлений для коэффициентов Фурье.

Вопрос 5: Как найти решение задачи Дирихле в круге с граничным условием \( f(\theta) = \cos(\theta) \)?

Ответ:

Подставляем \( f(\theta) = \cos(\theta) \) в формулу Пуассона и вычисляем интеграл. Решение имеет вид: \[ u(r, \theta) = r \cos(\theta). \]
