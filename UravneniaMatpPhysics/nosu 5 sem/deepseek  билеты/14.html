Билет № 14: Уравнения математической физики

1. Дивергенция векторного поля. Теорема Гаусса-Остроградского

Определение:

Векторное поле — это функция, которая каждой точке пространства ставит в соответствие вектор. Например, поле скоростей жидкости или поле сил.

Дивергенция векторного поля:

Дивергенция — это скалярная величина, которая характеризует "расходимость" векторного поля в данной точке. Для векторного поля \( \mathbf{F} = (F_x, F_y, F_z) \) дивергенция определяется как: \[ \text{div} \, \mathbf{F} = \frac{\partial F_x}{\partial x} + \frac{\partial F_y}{\partial y} + \frac{\partial F_z}{\partial z}. \]

Пример:

Рассмотрим векторное поле \( \mathbf{F} = (x^2, y^2, z^2) \). Найдем его дивергенцию: \[ \text{div} \, \mathbf{F} = \frac{\partial (x^2)}{\partial x} + \frac{\partial (y^2)}{\partial y} + \frac{\partial (z^2)}{\partial z} = 2x + 2y + 2z. \]

Теорема Гаусса-Остроградского:

Эта теорема связывает поток векторного поля через замкнутую поверхность с объемным интегралом от дивергенции этого поля внутри поверхности. Формально: \[ \iint_{\partial V} \mathbf{F} \cdot d\mathbf{S} = \iiint_V \text{div} \, \mathbf{F} \, dV, \] где:

  • \( \partial V \) — замкнутая поверхность, ограничивающая объем \( V \),
  • \( d\mathbf{S} \) — вектор элемента площади поверхности,
  • \( dV \) — элемент объема.

Пример задачи:

Вычислить поток векторного поля \( \mathbf{F} = (x, y, z) \) через сферу радиуса \( R \) с центром в начале координат.

Решение: 1. Найдем дивергенцию поля \( \mathbf{F} \): \[ \text{div} \, \mathbf{F} = \frac{\partial x}{\partial x} + \frac{\partial y}{\partial y} + \frac{\partial z}{\partial z} = 1 + 1 + 1 = 3. \] 2. Применим теорему Гаусса-Остроградского: \[ \iint_{\partial V} \mathbf{F} \cdot d\mathbf{S} = \iiint_V \text{div} \, \mathbf{F} \, dV = \iiint_V 3 \, dV. \] 3. Объем сферы радиуса \( R \) равен \( \frac{4}{3} \pi R^3 \), поэтому: \[ \iint_{\partial V} \mathbf{F} \cdot d\mathbf{S} = 3 \cdot \frac{4}{3} \pi R^3 = 4 \pi R^3. \]

2. Задача Коши для линейных дифференциальных уравнений с постоянными коэффициентами

Определение:

Линейное дифференциальное уравнение с постоянными коэффициентами — это уравнение вида: \[ a_n y^{(n)} + a_{n-1} y^{(n-1)} + \dots + a_1 y' + a_0 y = f(x), \] где:

  • \( y^{(k)} \) — \( k \)-я производная функции \( y(x) \),
  • \( a_n, a_{n-1}, \dots, a_0 \) — постоянные коэффициенты,
  • \( f(x) \) — заданная функция.

Задача Коши:

Это задача нахождения решения дифференциального уравнения, удовлетворяющего начальным условиям: \[ y(x_0) = y_0, \quad y'(x_0) = y_1, \quad \dots, \quad y^{(n-1)}(x_0) = y_{n-1}. \]

Решение:

1. Характеристическое уравнение: Для однородного уравнения (когда \( f(x) = 0 \)) решение ищется в виде \( y = e^{rx} \). Подставляя это в уравнение, получаем характеристическое уравнение: \[ a_n r^n + a_{n-1} r^{n-1} + \dots + a_1 r + a_0 = 0. \] 2. Общее решение однородного уравнения: Если корни характеристического уравнения \( r_1, r_2, \dots, r_n \) различны, то общее решение имеет вид: \[ y(x) = C_1 e^{r_1 x} + C_2 e^{r_2 x} + \dots + C_n e^{r_n x}. \] Если есть кратные корни, то решение включает дополнительные члены вида \( x^k e^{rx} \). 3. Частное решение неоднородного уравнения: Если \( f(x) \neq 0 \), то частное решение можно найти методом неопределенных коэффициентов или методом вариации постоянных. 4. Общее решение неоднородного уравнения: Сумма общего решения однородного уравнения и частного решения неоднородного уравнения. 5. Начальные условия: Используем начальные условия для нахождения констант \( C_1, C_2, \dots, C_n \).

Пример задачи:

Решить задачу Коши для уравнения: \[ y'' - 3y' + 2y = 0, \quad y(0) = 1, \quad y'(0) = 0. \]

Решение: 1. Характеристическое уравнение: \[ r^2 - 3r + 2 = 0. \] Корни: \( r_1 = 1 \), \( r_2 = 2 \). 2. Общее решение однородного уравнения: \[ y(x) = C_1 e^{x} + C_2 e^{2x}. \] 3. Находим производную: \[ y'(x) = C_1 e^{x} + 2 C_2 e^{2x}. \] 4. Подставляем начальные условия: \[ y(0) = C_1 + C_2 = 1, \] \[ y'(0) = C_1 + 2 C_2 = 0. \] 5. Решаем систему уравнений: \[ C_1 = 2, \quad C_2 = -1. \] 6. Итоговое решение: \[ y(x) = 2 e^{x} - e^{2x}. \]

Вопросы, которые могут возникнуть у преподавателя, и ответы на них:

Вопрос 1: Что такое дивергенция векторного поля?

Ответ:

Дивергенция — это скалярная величина, которая характеризует "расходимость" векторного поля в данной точке. Для векторного поля \( \mathbf{F} = (F_x, F_y, F_z) \) дивергенция определяется как: \[ \text{div} \, \mathbf{F} = \frac{\partial F_x}{\partial x} + \frac{\partial F_y}{\partial y} + \frac{\partial F_z}{\partial z}. \]

Вопрос 2: Что утверждает теорема Гаусса-Остроградского?

Ответ:

Теорема Гаусса-Остроградского связывает поток векторного поля через замкнутую поверхность с объемным интегралом от дивергенции этого поля внутри поверхности: \[ \iint_{\partial V} \mathbf{F} \cdot d\mathbf{S} = \iiint_V \text{div} \, \mathbf{F} \, dV. \]

Вопрос 3: Как решается задача Коши для линейного дифференциального уравнения с постоянными коэффициентами?

Ответ:

Решение задачи Коши включает нахождение общего решения однородного уравнения с помощью характеристического уравнения, нахождение частного решения неоднородного уравнения (если \( f(x) \neq 0 \)) и использование начальных условий для определения констант.

Вопрос 4: Как найти общее решение однородного дифференциального уравнения?

Ответ:

Общее решение однородного уравнения находится с помощью характеристического уравнения. Если корни \( r_1, r_2, \dots, r_n \) различны, то общее решение имеет вид: \[ y(x) = C_1 e^{r_1 x} + C_2 e^{r_2 x} + \dots + C_n e^{r_n x}. \]

Вопрос 5: Как использовать начальные условия для нахождения констант?

Ответ:

Начальные условия подставляются в общее решение и его производные, чтобы получить систему уравнений для констант \( C_1, C_2, \dots, C_n \).
