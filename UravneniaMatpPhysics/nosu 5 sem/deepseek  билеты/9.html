Билет № 9: Уравнения математической физики

1. Начально-краевая задача для неоднородного уравнения теплопроводности

Определение:

Уравнение теплопроводности описывает распределение температуры в теле с течением времени. Неоднородное уравнение теплопроводности имеет вид: \[ u_t = a^2 u_{xx} + f(x, t), \] где:

  • \( u(x, t) \) — температура в точке \( x \) в момент времени \( t \),
  • \( a \) — коэффициент теплопроводности,
  • \( f(x, t) \) — функция, описывающая внешний источник тепла.

Начально-краевая задача:

Это задача, в которой заданы:

  1. Начальное условие: \( u(x, 0) = \phi(x) \) — распределение температуры в начальный момент времени.
  2. Краевые условия: например, \( u(0, t) = \mu(t) \) и \( u(L, t) = \nu(t) \) — температура на границах стержня.

Пример:

Рассмотрим стержень длиной \( L \), на концах которого поддерживается нулевая температура, а начальное распределение температуры задано функцией \( \phi(x) \). Уравнение теплопроводности: \[ u_t = a^2 u_{xx}, \quad 0 < x < L, \quad t > 0. \] Начальное условие: \[ u(x, 0) = \phi(x). \] Краевые условия: \[ u(0, t) = 0, \quad u(L, t) = 0. \]

Решение:

1. Метод разделения переменных: Предположим, что решение можно представить в виде: \[ u(x, t) = X(x) T(t). \] 2. Подставляем это в уравнение теплопроводности: \[ X(x) T'(t) = a^2 X''(x) T(t). \] 3. Разделяем переменные: \[ \frac{T'(t)}{a^2 T(t)} = \frac{X''(x)}{X(x)} = -\lambda, \] где \( \lambda \) — постоянная разделения. 4. Получаем два обыкновенных дифференциальных уравнения: \[ X''(x) + \lambda X(x) = 0, \] \[ T'(t) + a^2 \lambda T(t) = 0. \] 5. Решаем уравнение для \( X(x) \): \[ X''(x) + \lambda X(x) = 0. \] Общее решение: \[ X(x) = A \cos(\sqrt{\lambda} x) + B \sin(\sqrt{\lambda} x). \] 6. Учитываем краевые условия \( X(0) = 0 \) и \( X(L) = 0 \): \[ X(0) = A = 0, \quad X(L) = B \sin(\sqrt{\lambda} L) = 0. \] Отсюда \( \sin(\sqrt{\lambda} L) = 0 \), поэтому: \[ \sqrt{\lambda} L = n \pi, \quad n = 1, 2, 3, \dots \] Таким образом, собственные значения: \[ \lambda_n = \left( \frac{n \pi}{L} \right)^2. \] Собственные функции: \[ X_n(x) = \sin\left( \frac{n \pi x}{L} \right). \] 7. Решаем уравнение для \( T(t) \): \[ T'(t) + a^2 \lambda_n T(t) = 0. \] Решение: \[ T_n(t) = C_n e^{-a^2 \lambda_n t}. \] 8. Общее решение: \[ u(x, t) = \sum_{n=1}^{\infty} C_n \sin\left( \frac{n \pi x}{L} \right) e^{-a^2 \left( \frac{n \pi}{L} \right)^2 t}. \] 9. Находим коэффициенты \( C_n \) из начального условия: \[ u(x, 0) = \phi(x) = \sum_{n=1}^{\infty} C_n \sin\left( \frac{n \pi x}{L} \right). \] Это ряд Фурье по синусам. Коэффициенты \( C_n \) вычисляются по формуле: \[ C_n = \frac{2}{L} \int_0^L \phi(x) \sin\left( \frac{n \pi x}{L} \right) dx. \]

Пример задачи:

Решить начально-краевую задачу для уравнения теплопроводности с начальным условием \( \phi(x) = x(L - x) \) и краевыми условиями \( u(0, t) = 0 \), \( u(L, t) = 0 \).

Решение: 1. Находим коэффициенты \( C_n \): \[ C_n = \frac{2}{L} \int_0^L x(L - x) \sin\left( \frac{n \pi x}{L} \right) dx. \] 2. Интегрируем по частям: \[ C_n = \frac{2}{L} \left[ \frac{2L^3}{n^3 \pi^3} (1 - (-1)^n) \right] = \frac{4L^2}{n^3 \pi^3} (1 - (-1)^n). \] 3. Подставляем \( C_n \) в общее решение: \[ u(x, t) = \sum_{n=1}^{\infty} \frac{4L^2}{n^3 \pi^3} (1 - (-1)^n) \sin\left( \frac{n \pi x}{L} \right) e^{-a^2 \left( \frac{n \pi}{L} \right)^2 t}. \]

2. Метод разделения переменных для нахождения решения краевой задачи колебаний струны

Определение:

Уравнение колебаний струны имеет вид: \[ u_{tt} = c^2 u_{xx}, \] где:

  • \( u(x, t) \) — отклонение струны от положения равновесия в точке \( x \) в момент времени \( t \),
  • \( c \) — скорость распространения волны.

Краевая задача:

Рассмотрим струну длиной \( L \), закрепленную на концах. Начальные условия: \[ u(x, 0) = \phi(x), \quad u_t(x, 0) = \psi(x). \] Краевые условия: \[ u(0, t) = 0, \quad u(L, t) = 0. \]

Решение:

1. Метод разделения переменных: Предположим, что решение можно представить в виде: \[ u(x, t) = X(x) T(t). \] 2. Подставляем это в уравнение колебаний струны: \[ X(x) T''(t) = c^2 X''(x) T(t). \] 3. Разделяем переменные: \[ \frac{T''(t)}{c^2 T(t)} = \frac{X''(x)}{X(x)} = -\lambda, \] где \( \lambda \) — постоянная разделения. 4. Получаем два обыкновенных дифференциальных уравнения: \[ X''(x) + \lambda X(x) = 0, \] \[ T''(t) + c^2 \lambda T(t) = 0. \] 5. Решаем уравнение для \( X(x) \): \[ X''(x) + \lambda X(x) = 0. \] Общее решение: \[ X(x) = A \cos(\sqrt{\lambda} x) + B \sin(\sqrt{\lambda} x). \] 6. Учитываем краевые условия \( X(0) = 0 \) и \( X(L) = 0 \): \[ X(0) = A = 0, \quad X(L) = B \sin(\sqrt{\lambda} L) = 0. \] Отсюда \( \sin(\sqrt{\lambda} L) = 0 \), поэтому: \[ \sqrt{\lambda} L = n \pi, \quad n = 1, 2, 3, \dots \] Таким образом, собственные значения: \[ \lambda_n = \left( \frac{n \pi}{L} \right)^2. \] Собственные функции: \[ X_n(x) = \sin\left( \frac{n \pi x}{L} \right). \] 7. Решаем уравнение для \( T(t) \): \[ T''(t) + c^2 \lambda_n T(t) = 0. \] Решение: \[ T_n(t) = C_n \cos\left( \frac{n \pi c t}{L} \right) + D_n \sin\left( \frac{n \pi c t}{L} \right). \] 8. Общее решение: \[ u(x, t) = \sum_{n=1}^{\infty} \left[ C_n \cos\left( \frac{n \pi c t}{L} \right) + D_n \sin\left( \frac{n \pi c t}{L} \right) \right] \sin\left( \frac{n \pi x}{L} \right). \] 9. Находим коэффициенты \( C_n \) и \( D_n \) из начальных условий: \[ u(x, 0) = \phi(x) = \sum_{n=1}^{\infty} C_n \sin\left( \frac{n \pi x}{L} \right), \] \[ u_t(x, 0) = \psi(x) = \sum_{n=1}^{\infty} D_n \frac{n \pi c}{L} \sin\left( \frac{n \pi x}{L} \right). \] Коэффициенты \( C_n \) и \( D_n \) вычисляются по формулам: \[ C_n = \frac{2}{L} \int_0^L \phi(x) \sin\left( \frac{n \pi x}{L} \right) dx, \] \[ D_n = \frac{2}{n \pi c} \int_0^L \psi(x) \sin\left( \frac{n \pi x}{L} \right) dx. \]

Пример задачи:

Решить краевую задачу для уравнения колебаний струны с начальными условиями \( \phi(x) = \sin\left( \frac{\pi x}{L} \right) \) и \( \psi(x) = 0 \).

Решение: 1. Находим коэффициенты \( C_n \): \[ C_n = \frac{2}{L} \int_0^L \sin\left( \frac{\pi x}{L} \right) \sin\left( \frac{n \pi x}{L} \right) dx. \] Для \( n = 1 \): \[ C_1 = \frac{2}{L} \int_0^L \sin^2\left( \frac{\pi x}{L} \right) dx = 1. \] Для \( n > 1 \): \[ C_n = 0. \] 2. Коэффициенты \( D_n = 0 \), так как \( \psi(x) = 0 \). 3. Подставляем \( C_n \) и \( D_n \) в общее решение: \[ u(x, t) = \cos\left( \frac{\pi c t}{L} \right) \sin\left( \frac{\pi x}{L} \right). \]

Вопросы, которые могут возникнуть у преподавателя, и ответы на них:

Вопрос 1: Что такое начально-краевая задача для уравнения теплопроводности?

Ответ:

Это задача, в которой заданы начальное распределение температуры и краевые условия (температура на границах тела). Решение описывает, как температура изменяется со временем.

Вопрос 2: Как работает метод разделения переменных?

Ответ:

Метод разделения переменных предполагает, что решение можно представить в виде произведения функций, каждая из которых зависит только от одной переменной. Это позволяет свести уравнение в частных производных к системе обыкновенных дифференциальных уравнений.

Вопрос 3: Как найти коэффициенты \( C_n \) и \( D_n \) в решении задачи колебаний струны?

Ответ:

Коэффициенты \( C_n \) и \( D_n \) находятся из начальных условий с помощью разложения в ряд Фурье по синусам.

Вопрос 4: Что такое собственные значения и собственные функции?

Ответ:

Собственные значения — это значения параметра \( \lambda \), при которых краевая задача имеет нетривиальное решение. Собственные функции — это соответствующие решения краевой задачи.

Вопрос 5: Как выглядит общее решение уравнения колебаний струны?

Ответ:

Общее решение имеет вид суммы гармонических функций, каждая из которых описывает стоячую волну на струне.
