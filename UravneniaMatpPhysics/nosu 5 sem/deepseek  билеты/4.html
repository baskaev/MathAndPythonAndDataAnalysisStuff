Билет № 4: Уравнения математической физики

1. Вывод уравнения колебаний мембраны

Что такое мембрана?

Мембрана — это тонкая, гибкая, упругая поверхность, которая может колебаться. Колебания мембраны описываются двумерным волновым уравнением.

Физические предположения:

  1. Мембрана однородная (плотность \( \rho \) постоянна).
  2. Натяжение \( T \) постоянно по всей мембране.
  3. Отклонения мембраны малы, поэтому углы наклона малы.

Уравнение колебаний мембраны:

\[ \frac{\partial^2 u}{\partial t^2} = c^2 \left( \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} \right), \]

где:

  • \( u(x, y, t) \) — отклонение мембраны от положения равновесия,
  • \( c = \sqrt{\frac{T}{\rho}} \) — скорость распространения волн в мембране.

Вывод уравнения:

  1. Силы, действующие на элемент мембраны: Рассмотрим малый элемент мембраны площадью \( \Delta x \Delta y \). На этот элемент действуют силы натяжения \( T \) по краям.
  2. Сумма вертикальных сил: Вертикальная сила, действующая на элемент мембраны, равна:

    \[ T \left( \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} \right) \Delta x \Delta y. \]

  3. Ускорение элемента мембраны: По второму закону Ньютона:

    \[ \rho \Delta x \Delta y \frac{\partial^2 u}{\partial t^2} = T \left( \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} \right) \Delta x \Delta y. \]

  4. Упрощение уравнения: Разделим на \( \rho \Delta x \Delta y \) и обозначим \( c^2 = \frac{T}{\rho} \):

    \[ \frac{\partial^2 u}{\partial t^2} = c^2 \left( \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} \right). \]

    Это и есть уравнение колебаний мембраны.

2. Распространение тепла в ограниченных телах. Схема метода разделения переменных

Что такое уравнение теплопроводности?

Уравнение теплопроводности описывает распределение температуры в теле с течением времени. Для ограниченного тела оно имеет вид:

\[ \frac{\partial u}{\partial t} = \alpha \left( \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} + \frac{\partial^2 u}{\partial z^2} \right), \]

где:

  • \( u(x, y, z, t) \) — температура в точке \( (x, y, z) \) в момент времени \( t \),
  • \( \alpha \) — коэффициент теплопроводности.

Метод разделения переменных:

Метод разделения переменных позволяет найти решение уравнения теплопроводности в виде произведения функций, каждая из которых зависит только от одной переменной.

Пример: Одномерное уравнение теплопроводности:

Рассмотрим одномерное уравнение теплопроводности:

\[ \frac{\partial u}{\partial t} = \alpha \frac{\partial^2 u}{\partial x^2}. \]

Граничные условия:

  1. На левой границе (\( x = 0 \)):

    \[ u(0, t) = 0. \]

  2. На правой границе (\( x = L \)):

    \[ u(L, t) = 0. \]

Начальное условие:

\[ u(x, 0) = f(x). \]

Решение методом разделения переменных:

  1. Предположим, что решение имеет вид:

    \[ u(x, t) = X(x) T(t). \]

  2. Подставим в уравнение:

    \[ X(x) T'(t) = \alpha X''(x) T(t). \]

  3. Разделим на \( X(x) T(t) \):

    \[ \frac{T'(t)}{\alpha T(t)} = \frac{X''(x)}{X(x)} = -\lambda. \]

  4. Получаем два обыкновенных дифференциальных уравнения:
    • Для \( X(x) \):

      \[ X''(x) + \lambda X(x) = 0. \]

    • Для \( T(t) \):

      \[ T'(t) + \alpha \lambda T(t) = 0. \]

  5. Решим уравнение для \( X(x) \): Общее решение:

    \[ X(x) = A \sin(\sqrt{\lambda} x) + B \cos(\sqrt{\lambda} x). \]

    Из граничных условий \( X(0) = 0 \) и \( X(L) = 0 \) получаем:

    \[ \sqrt{\lambda} = \frac{n \pi}{L}, \quad n = 1, 2, 3, \dots \]

    Таким образом:

    \[ X_n(x) = \sin\left(\frac{n \pi x}{L}\right). \]

  6. Решим уравнение для \( T(t) \): Общее решение:

    \[ T_n(t) = C_n e^{-\alpha \left(\frac{n \pi}{L}\right)^2 t}. \]

  7. Общее решение уравнения теплопроводности:

    \[ u(x, t) = \sum_{n=1}^{\infty} C_n \sin\left(\frac{n \pi x}{L}\right) e^{-\alpha \left(\frac{n \pi}{L}\right)^2 t}. \]

  8. Определим коэффициенты \( C_n \): Используем начальное условие \( u(x, 0) = f(x) \):

    \[ f(x) = \sum_{n=1}^{\infty} C_n \sin\left(\frac{n \pi x}{L}\right). \]

    Коэффициенты \( C_n \) находятся по формуле:

    \[ C_n = \frac{2}{L} \int_0^L f(x) \sin\left(\frac{n \pi x}{L}\right) dx. \]

Итог

1. Вывод уравнения колебаний мембраны:

  • Уравнение: \( \frac{\partial^2 u}{\partial t^2} = c^2 \left( \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} \right) \).
  • Физические предположения.
  • Вывод уравнения.

2. Распространение тепла в ограниченных телах:

  • Уравнение теплопроводности: \( \frac{\partial u}{\partial t} = \alpha \frac{\partial^2 u}{\partial x^2} \).
  • Метод разделения переменных.
  • Решение с граничными и начальными условиями.
